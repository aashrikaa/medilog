🤔 JavaScript If Statements

What are If Statements? 🤔

If statements are like decision makers in your code. They let your program choose different paths based on conditions!

Think of it like this: "If it's raining, take an umbrella. Otherwise, leave it at home."

How If Statements Work 🔄

Check Condition
→
Is it True?
→
Run Code

The program checks if a condition is true, and if it is, it runs the code inside the if statement.

Basic If Statement 📝

Here's the simplest form:

if (condition) {
    // This code runs if condition is true
    console.log("Condition is true!");
}

The condition is usually a comparison that results in true or false.

If-Else Statements 🔀

Sometimes you want to do one thing if a condition is true, and something else if it's false:

if (age >= 18) {
    console.log("You can vote!");
} else {
    console.log("Too young to vote.");
}

The else block runs when the condition is false.

Else-If Statements 🔀🔀

For multiple conditions, use else if:

if (score >= 90) {
    grade = "A";
} else if (score >= 80) {
    grade = "B";
} else if (score >= 70) {
    grade = "C";
} else {
    grade = "F";
}

This checks conditions in order until one is true.

Common Conditions 🔍

Here are some typical conditions you'll use:

// Comparisons
age >= 18; // Greater than or equal to
temperature < 0; // Less than
name === "John"; // Exactly equal to
password != ""; // Not equal to

// Logical combinations
age >= 18 && hasID; // Both must be true
isStudent || isTeacher; // Either can be true
!isLoggedIn; // Opposite of isLoggedIn
