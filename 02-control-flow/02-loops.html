🔄 JavaScript Loops

What are Loops? 🤔

Loops are like repetition machines in your code. They let you run the same code multiple times without writing it over and over!

Think of it like this: "Count from 1 to 10" or "Print 'Hello' 5 times"

How Loops Work 🔄

↻

Loops keep running until a condition becomes false. The arrow going back shows the repetition!

For Loop 📝

The most common loop - great for counting and going through lists:

for (let i = 0; i < 5; i++) {
    console.log("Count: " + i);
}

This has three parts:

  1. let i = 0 - Start with 0
  2. i < 5 - Keep going while i is less than 5
  3. i++ - Add 1 to i each time

While Loop 🔄

Keeps running while a condition is true:

let count = 0;
while (count < 5) {
    console.log("Count: " + count);
    count++;
}

Important: Make sure the condition eventually becomes false, or you'll have an infinite loop!

Do-While Loop 🔄

Similar to while, but runs at least once:

let number = 1;
do {
    console.log("Number: " + number);
    number++;
} while (number <= 3);

This will always run at least once, even if the condition is false from the start.

Loop Control 🛑

Sometimes you want to control the loop flow:

for (let i = 1; i <= 10; i++) {
    if (i === 5) {
        continue; // Skip 5
    }
    if (i === 8) {
        break; // Stop at 8
    }
    console.log(i);
}

continue skips to the next iteration, break exits the loop entirely.
