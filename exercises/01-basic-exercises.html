Exercise 3: Age Checker 🔍

Create a program that checks if someone is old enough to drive (16+) and vote (18+).

// Your code goes here
// 1. Get age from user
// 2. Use if statements to check driving and voting age
// 3. Display appropriate messages
Enter an age to check driving and voting eligibility!
Solution:
let age = 17;

if (age >= 18) {
    console.log("You can drive and vote!");
} else if (age >= 16) {
    console.log("You can drive (with restrictions) but can't vote.");
} else {
    console.log("You're too young to drive or vote.");
}

// Your code goes here
// 1. Get a number from user

}

Exercise 5: Simple Calculator 🧮

Create a function that takes two numbers and an operation (+, -, *, /) and returns the result.

// Your code goes here
// 1. Create a function with three parameters
// 2. Use if statements to handle different operations
// 3. Return the result
Enter numbers and select an operation!
Solution:
function calculate(num1, num2, operation) {
    if (operation === '+') return num1 + num2;
    if (operation === '-') return num1 - num2;
    if (operation === '*') return num1 * num2;
    if (operation === '/') return num1 / num2;
    return "Invalid operation";
}

let result = calculate(10, 5, '+');
console.log("Result:", result);
