🔢 JavaScript Operators

What are Operators? 🤔

Operators are symbols that perform operations on values. Think of them as tools that help you work with data!

JavaScript has several types of operators:

  • Arithmetic Operators - for math calculations
  • Assignment Operators - for storing values
  • Comparison Operators - for comparing values
  • Logical Operators - for combining conditions

Assignment Operators 📝

These operators assign values to variables:

let x = 5; // Basic assignment
x += 3; // Same as: x = x + 3 (result: 8)
x -= 2; // Same as: x = x - 2 (result: 6)
x *= 4; // Same as: x = x * 4 (result: 24)
x /= 6; // Same as: x = x / 6 (result: 4)
x %= 3; // Same as: x = x % 3 (result: 1)

Comparison Operators 🔍

These operators compare values and return true or false:

let a = 5, b = 10;

a == b; // false (equal value)
a === b; // false (equal value AND type)
a != b; // true (not equal)
a < b; // true (less than)
a <= b; // true (less than or equal)
a > b; // false (greater than)
a >= b; // false (greater than or equal)

Logical Operators 🧠

These operators combine multiple conditions:

let isStudent = true;
let hasMoney = false;

isStudent && hasMoney; // false (AND - both must be true)
isStudent || hasMoney; // true (OR - at least one must be true)
!isStudent; // false (NOT - opposite of true)

Practice Examples 💻

Try these in your browser console:

// Arithmetic
let result = (10 + 5) * 2;
console.log("Result:", result);

// Comparison
let age = 18;
let canVote = age >= 18;
console.log("Can vote?", canVote);

// Logical
let hasLicense = true;
let hasCar = false;
let canDrive = hasLicense && hasCar;
console.log("Can drive?", canDrive);
