🎯 JavaScript Variables

What is a Variable? 🤔

A variable is like a container that stores information. Think of it as a labeled box where you can put different things!

In JavaScript, we create variables using three keywords:

  • let - for variables that can change
  • const - for variables that stay the same
  • var - older way (we'll use the modern ones!)

Creating Your First Variable 🚀

Let's create a variable to store your name:

let myName = "Your Name Here";

This line does three things:

  1. let - declares a new variable
  2. myName - gives the variable a name
  3. "Your Name Here" - assigns a value to it

Different Types of Data 📊

JavaScript variables can store different types of information:

  • Strings - text (like "Hello World")
  • Numbers - whole numbers (42) or decimals (3.14)
  • Booleans - true or false
  • Undefined - when a variable has no value
  • Null - intentionally empty

Naming Rules 📝

When naming variables, follow these rules:

  • Start with a letter, underscore (_), or dollar sign ($)
  • Can contain letters, numbers, underscores, and dollar signs
  • Cannot use JavaScript keywords (like let, const, if, etc.)
  • Use camelCase for multiple words (myFirstName)
  • Make names descriptive and meaningful
