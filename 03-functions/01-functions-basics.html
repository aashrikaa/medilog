🔧 JavaScript Functions

What are Functions? 🤔

Functions are like mini-programs inside your code. They're reusable blocks of code that do specific tasks!

Think of them like recipes: you write the recipe once, then you can use it many times to cook the same dish.

How Functions Work 🔄

Input (Parameters)
→
Function Body
→
Output (Return)

Functions take inputs, process them, and give you back results. Like a machine that transforms things!

Creating Your First Function 📝

Here's how to create a simple function:

function sayHello() {
    console.log("Hello, World!");
}

// To use the function:
sayHello(); // This calls the function

This function has:

  1. function - keyword to declare a function
  2. sayHello - the function's name
  3. () - parentheses (empty for now)
  4. {} - curly braces containing the code

Functions with Parameters 📥

Parameters are like inputs to your function:

function greet(name) {
    console.log("Hello, " + name + "!");
}

greet("Alice"); // Outputs: "Hello, Alice!"
greet("Bob"); // Outputs: "Hello, Bob!"

The name parameter acts like a variable that gets its value when you call the function.

Functions with Return Values 📤

Functions can give you back results:

function add(a, b) {
    return a + b;
}

let result = add(5, 3); // result = 8
console.log(result); // Outputs: 8

The return keyword sends a value back to wherever the function was called from.

Function Types 🔀

JavaScript has several ways to create functions:

// Function Declaration
function multiply(x, y) {
    return x * y;
}

// Function Expression
let divide = function(a, b) {
    return a / b;
};

// Arrow Function (ES6+)
let subtract = (x, y) => x - y;

All three do the same thing, just written differently!

Practice Examples 💻

Try these in your browser console:

// Calculate area of a circle
function circleArea(radius) {
    return Math.PI * radius * radius;
}

let area = circleArea(5);
console.log("Area:", area);

// Check if number is even
function isEven(num) {
    return num % 2 === 0;
}

console.log("Is 8 even?", isEven(8));
console.log("Is 7 even?", isEven(7));
